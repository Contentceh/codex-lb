1) Запустить браузер из сборки складчины, нажать F12, перейти на вкладку console
2) Вставить скрипт из файла portable_ripper.txt и нажать enter . На этом шаге может активироваться защита браузера и он попросит набрать фразу в консоли - ввести фразу и снова вставить скрипт
3) Скрипт выплюнет данные, нас интересуют: access_token, session_token, email, еще есть дата истечения аккаунта(для ведения записей когда акки закончаться)
4) Запустить session_importer.py, он по очереди запросит access_token, session_token, email(без кавычек!!), делаем копировать-вставить из браузера, результат будет - файл auth_текущая_дата.json
5) Заходим в админку codex-lb - accounts. Жмём Import, загружаем файл auth_текущая_дата.json
Готово.
